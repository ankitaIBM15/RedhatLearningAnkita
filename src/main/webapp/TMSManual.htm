Train Monitoring System (TMS)

 

Train Monitoring System (TMS) is web-based software that can be used to quickly identify active trains that meet specific criteria with respect to last reported division and/or train delay.

TMS has the ability to provide additional information such as train consist, ETAs, train event history, and to show the current locations of trains on a map.

Accessing the Application

Open Internet Explorer and go to http://nsight.nscorp.com/locomotiveSystems. This is the Locomotive Systems Main Menu Page. There are 2 ways to get to the TMS application.

1. Mouse over the Locomotive Executive Info System (EIS) and click on the Train Monitoring System (TMS)
2. Click on the Locomotive Executive Info System (EIS).This will take you to the Locomotive EIS Main Page. Click on the TMS link.

If your computer has been upgraded with the required plug-in(preferred Java 1.4 plugin) , you will be taken directly to TMS. Your browser will load TMS and it will appear as a separate window on your screen. If you do not have the Java plug-in, a window will instruct you to call NSS and ask them to create a ticket for PCLan to install the plug-in.

TMS Main Screen

This screen is where the user will do all of their searches for trains. The user has the ability to search for a particular train symbol, all trains at a particular origin or destination, all trains on a division, a preset list of train symbols, or the user’s predefined train list. To further narrow down the trains returned, the user can use the check boxes to show a list of trains with the specified criteria.  Users can store customized train lists and profiles if they have a valid Identity Services password (also known as an LDAP password).  Users must contact their security point-of-contact (SPOC) for Identity Services access or password resets.  Refer to Additional Functionality Available by Logging into TMS for more information about the customization features.

There are two different views for this application:  the Transportation view and the Intermodal view.  Both views return the same trains, but they return different information about the trains.

Functionality Available without Logging into TMS

Searching for Trains

Click on the rectangle that is titled "Train List." When you select a new train list, the application will rebuild the train symbols drop down list with all of the train symbols assigned to the selected train list. Select a specific train from the train symbols or leave the default selected (all trains) and click on ‘Fetch Data’. This will retrieve all active trains for the specified list or the specific train. The application will display the train symbol, status (Ordered, On Time, Behind, Late, or No Rept), whether a train is relative to its schedule, the time of the last reporting, the station of the last reporting, the event type last reported, number of loaded cars, number of empty cars, total weight of the train (in tons), length of train, the train’s originating station, and the train’s destination station. Any train that has a status of late is more than 2 hours behind schedule. Trains with the status of No Rprt have not had a reporting within the last 6 hours. The user can sort by any of the columns by clicking on the column header or rearrange the columns by clicking and holding onto the column and moving to the desired location. The user can also sort in descending order by holding the shift key down and clicking on the column header.

For InterModal View the user should first check the "InterModal View" check box and then click the 'Fetch Data' button. The application will display the train number, ETA (Estimated Time of Arrival) , delay information, the time of the last reporting, the location of the last reporting, the event type last reported, number of loaded cars, number of empty cars, number of stack cars,length of the train, train's origin station and the train's destination station.

The user can drill down on each train to get further details such as Schedule, Event History, Consist, Map, InterModal Scanner and Excel. Clicking on Schedule will bring up a new window displaying the train’s schedule and other data related to the schedule. Clicking on Event History will show each reporting, scanner and CTC event that the train has had from its origin. Consist will show all locomotives and railcars reported on the train. InterModal Scanner Details will show the sequence, car, equipment and chas. initial and numbers, loaded or empty and other details related to scanner. To Excel option will transfer the train details to an Excel file which can be saved on the local PC. The user can sort by any of the columns by clicking on the column header or rearrange the columns by clicking and holding onto the column and moving to the desired location. Finally, the user can select the Map option and display the train’s location on a map.

The user can search for trains with a specified criteria. The various criteria provided by the application are

  • Train Symbol - Will display all the trains which has the selected train symbol
  • Train Lists - Will display all trains which are in the selected train list
  • Origin - Will display all trains which has the selected origin station name
  • Destination - Will display all the trains which has the selected destination station name
  • Division - Will display all the trains which are in the selected division
  • Train Status - Will display all the trains of the selected status
  • Minimum Delay - Will display all the trains which has had a minimum delay of the specified hours

Map Button

After a list of trains has been retrieved, the user can click on the ‘Map’ button to display all trains on the list. From there, the user can zoom in on the particular area they are interested in viewing. They can use the ‘Zoom In’ button or click on the map and trace an area by holding down the mouse button and moving the mouse. This will zoom into the area they have just drawn. Trains are shown with circles and train symbols, stations are denoted with gray triangles, track is shown in red, and roads shown in green. By clicking on the boxes on the left-hand side, the user can remove or add any of the features displayed on the map. While zoomed in, the user can click on the ‘Pan’ button and move the map in any direction. Simply click and hold the mouse button and drag the image in the opposite direction you want to see. This allows users to move around at the zoomed in level instead of zooming out to the top view and reselecting the area they want to view.

Print Button

The print button allows users to print the list of trains loaded on the screen.

Help Button

The Help button provides information to the users on whom to contact, if they have problems with the application

 

Additional Functionality Available by Logging into TMS

Users who log into TMS will be allowed to create custom train lists and user profiles in addition to the basic functionality.

Login

Click on the Login button to access the login screen. Type in your racfid and your Identity Services(LDAP) password and hit enter on your keyboard. This will now allow the user to click on the User Profiles, New Train Lists, and Edit Train Lists buttons.

New Train Lists

Clicking on the New Train List button will give the user the screen where they can create a customized list of train symbols. The user can select trains from any list from the dropdown on the left-hand side and add it to their list. To add a train to the new list, click on the train to highlight it and click the right arrow button. Hold down the control key on the keyboard and click on trains to select multiple trains and click the right arrow button. Create a name for the train list and enter it into the ‘Train List Name’ space. Click on ‘Save & Exit’.

Click on the ‘Train List’ dropdown list and scroll to the bottom to find your new custom train list.

Edit Train Lists

To edit an existing custom train list, select the list from the ‘Train List’ dropdown menu . Next click on ‘Edit Train Lists’ which will pull up a new window. To delete a train from the list, click on the train and click the ç button to move it from the right side to the left side. This will remove an individual train. Adding new trains is the same as when you created the list. Select from the dropdown list on the left and use the right arrow button to add new trains. Click ‘Update & Exit’ when finished. The user may also delete the whole train list by clicking on the ‘Delete Train List’ button.

User Profiles

User profiles allow the user to create a profile with a train list and a particular set of search criteria. Clicking on ‘User Profiles’ brings up the screen to create these profiles. The user selects a name for the profile, what train list they want to use, and any search criteria to filter by. For example, a user can create a profile that shows all the Intermodal trains that are late in the Georgia division. Once this is created, the user simply selects this profile from the ‘User Profile’ dropdown list and clicks ‘Fetch Data’. The search criteria will automatically be filled in.
